eGuide is a module for Event management.

## Description
* eguide is a powerful Xoops registration module, with flexible custom input of various forms, registration of multiple numbers, registration approval mechanism, automatic or manual sending mechanism, repeated open registration, required field form verification, front desk flexibility Display registration field data, automatic statistics, export CSV or Excel format registration information ... And other functions
* The original author of this module is nbuy (https://github.com/nbuy)
* This revised version of the template applied bootstrap3, your background needs to support bootstrap3, for example: xBootstrap (https://xoops.org/modules/news/article.php?storyid=6571)
* Create and modify registration, date picker bootstrap-datepicker (https://github.com/eternicode/bootstrap-datepicker), WYSIWYG editor TinyMCE4 (http://www.tinymce.com/)


## Usage
* Download Unzip the name of the module folder from xoops-modules-eguide to eguide after installation, installation and general Xoops module the same
* Enable TinyMCE, go to use_fckeditor = default as input
* Upgrade users, activity status marked to be set to

0,<span class="label label-default">Close</span>,50,<span class="label label label-success">Registered</span>,100,<span class="label label-warning">Confirm</span>,101,<span class="label label-danger">Full</span>
