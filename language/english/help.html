Additional fileld Formats

 In Reservation Additonal items set to FORM when use order to reservation. This additional items to setup following formats:
item-name["*"][,type[,argument...]]

type ::= {text|checkbox|radio|textarea|select}
argument ::= [value ["+"] "="] label

  • "*" meen need input something.
  • "#" is message in the FORM (displayed comments).
  • "," is derimiter. (Use HTML special char when use in arguments)
  • value equivarent label when it suppressed.
  • "+" Selected items (checkbox, radio, select) default value.

examples:
