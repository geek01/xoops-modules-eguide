Summary

 Event guide module will be display event guidance and reservations the envet. The modules have functions as followed:
  • Event reservations
  • Configurable reservation forms - register handle by email address
  • Display event automatical in time - change display and allow reserve
  • Adminstration reservations and get data CSV format
  • Mail to send reserved person
  • New event notify to registers

Using Steps
  1. Install this module
    2. XOOPS at [Admin Menu - System - Modules Admin] to install "event guide" module.
    module setup for event guide if you want. [Admin Menu - Event Guide - module setup]
  2. Register New event
    3. Write event guide document and register it. The reservation form can change after.
  3. Event reservations
    4. Reservation list and Adminstration can login permition user at "Adminstrations" link.

Additional fileld Formats

 In Reservation Additonal items set to FORM when use order to reservation. This additional items to setup following formats:
line ::= ["!"]item-name["*"][,type[,argument...]]

type ::= {text|checkbox|radio|textarea|select}
argument ::= [value ["+"] "="] label

  • "!" meen the item display public.
    • Additional form top of line marked "!" reservation list are display. There is no "!" marked items, the list not display.
  • "*" meen need input something.
  • "#" is message in the FORM (displayed comments).
  • "," is derimiter. (Use HTML special char when use in arguments)
  • value equivarent label when it suppressed.
  • "+" Selected items (checkbox, radio, select) default value.

examples:

setupDisplayed form
Name*Name*
Name*,size=5Name*
Name*,size=5,#comment textName* comment text
Selection,radio,Item1+,Item2,Item3Selection Item1   Item2   Item3  
Selection,checkbox,Item1+,Item2,Item3Selection Item1   Item2   Item3  
Selection,select,Item1,Item2,Item3Selection
#message textmessage text
